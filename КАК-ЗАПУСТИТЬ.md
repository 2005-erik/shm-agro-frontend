# Сайт ТОО «СХМ Агро» — прототип

Одностраничный сайт (SPA) производителя сельхозтехники. Работает офлайн — открывается
двойным кликом по `index.html`. Для AI-чата нужен отдельный бэкенд (см. ниже).

## Как открыть

Двойной клик по `index.html` → откроется в браузере. Всё работает офлайн,
кроме карты (нужен интернет) и AI-чата (нужен запущенный бэкенд).

## AI-чат-бот

Кнопка 💬 справа снизу. Отвечает двумя способами:
- **Если запущен бэкенд** (папка `shm-agro-backend`) — через AI (Groq)
- **Если бэкенд выключен** — по локальной базе (без AI, чат не ломается)

Чтобы включить AI: запусти бэкенд из папки `shm-agro-backend`
(инструкция в его `README.md`). Нужен ключ Groq в файле `.env`.

## Где что править в index.html

- `models: [ ... ]` — техника каталога (название, фото, галерея, характеристики)
- `news: [ ... ]` — новости (заголовок, дата, обложка, текст)
- `certs: [ ... ]` — сертификаты
- Контакты (телефон, адрес, почта) — в футере и блоке «Контакты», ищи через Cmd+F
- Карта — блок «Контакты», iframe с `yandex.ru/map-widget` (меняется параметр `text=`)
- Чат-бот — адрес бэкенда в переменной `BACKEND_URL` (для деплоя поменять на адрес сервера)

## Прототип

Данные (техника, новости, сертификаты, адрес) — демонстрационные, для предпросмотра.
Перед запуском в прод заменить на реальные материалы заказчика.
